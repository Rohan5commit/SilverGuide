# Senior Center Pilot

## How a senior center could use SilverGuide

SilverGuide is designed for simple, repeatable workshops. A center can run it as a 20 to 30 minute session on a shared screen, on a few tablets, or on laptop stations.

Suggested session flow:

1. Start on the homepage and explain the four modules.
2. Run one Scam Shield example together as a group.
3. Let learners try one Everyday Tech Coach lesson on their own device or with a volunteer beside them.
4. End with Practice Mode and review the summary aloud.

## Hardware needed

- A laptop, desktop, or tablet with a modern browser
- Internet access if the center wants live NVIDIA NIM features
- Speakers or headphones if using the built-in voice playback

No special hardware is required.

## How volunteers can support learners

- Read the screen aloud if a learner wants extra support.
- Encourage learners to say why they chose an answer before revealing the explanation.
- Use the Back and Next buttons to keep the pace slow and comfortable.
- Turn on large-text mode when a learner needs easier reading.
- Focus on one skill at a time instead of jumping between modules.

## Why setup is simple

- No login is required for the MVP.
- Built-in lessons and scam examples are already in the repo.
- The browser speech API handles the listen buttons without extra services.
- The app can still teach useful material even when NVIDIA NIM is unavailable because fallback content is included.

## Best pilot format

- Group size: 5 to 15 learners
- Session length: 20 to 30 minutes
- Staff needed: 1 facilitator, plus 1 volunteer for every few learners if possible
- Recommended first session: Scam Shield + Joining a video call + Practice Mode

## What makes the pilot realistic

SilverGuide is not a toy chatbot. It is a structured teaching tool with clear lessons, repeatable scenarios, and a calm interface that matches the pace many older adults prefer when learning technology.
